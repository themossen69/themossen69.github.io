# README

Pàgina personal d'Àjan Perarnau Pagès aka themossen69. Projecte creat amb l'objectiu d'aprendre de pàgines web.

## Accés

Accedeix a la web amb l'enllaç [ajan.perarnau.cat](https://ajan.perarnau.cat/)

## Notícies
Blog personal de _themossen69_.

Funciona amb `jekyll` (ja configurat amb _github pages_) de manera que només creant un fitxer markdown a la carpeta `_posts` aquest es penja automaticament com a notícia a la web formatejat amb la plantilla de `./notícies/index.html`.

## Paraulari
Secció de la pàgina web on pots enriquir el teu vocabulari en català amb paraules ben curioses. Entra a l'apartat **diccionari** per conèixer els mots i posa't a prova amb el joc **la parelleta** relacionant-los amb la seva definició!

Funciona a través de scripts de javascript amb el contingut del fitxer `words.json`.
